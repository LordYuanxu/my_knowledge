## 查看信息

```r
# 查看包名
.packages(all.available = TRUE)

# 查看详细信息（含版本、路径、依赖等）
installed.packages()

# 查看已加载的包
sessionInfo()

# 查看包安装在哪里
.libPaths()

# 查看某个包的具体位置
find.package("ggplot2")
```

## conda搜索和安装

```bash
# 搜索 CRAN 包（前缀通常是 r-）
conda search r-包名

# 搜索 Bioconductor 包（前缀通常也是 r-）
conda search r-deseq2

# 或者用 mamba 更快
mamba search r-包名
```

## 批量清理孤儿依赖(使用pak包)

```r
pak::pkg_deps_tree("包名")
pkg::pkg_remove("包名")
```
