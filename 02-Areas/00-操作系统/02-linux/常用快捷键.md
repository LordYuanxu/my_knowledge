# 删除相关

- ‌**Ctrl + H**‌：删除光标‌**前**‌一个字符（等同退格键 Backspace）
- ‌**Ctrl + D**‌：删除光标‌**后**‌一个字符（若行为空则可能退出 Shell）
- ‌**Ctrl + W**‌：删除光标前‌**一个单词**‌（以空格/标点为界）
- ‌**Ctrl + U**‌：删除光标‌**至行首**‌的所有内容
- ‌**Ctrl + K**‌：删除光标‌**至行尾**‌的所有内容
- ‌**Alt + D**‌：删除光标‌**后**‌一个单词

# 移动相关

- ‌**Ctrl + A**‌：跳转到行首
- ‌**Ctrl + E**‌：跳转到行尾
- ‌**Ctrl + F**‌ / ‌**→**‌：向右移动一个字符
- ‌**Ctrl + B**‌ / ‌**←**‌：向左移动一个字符
- ‌**Alt + F**‌（或 ‌**Esc + F**‌）：向右跳过一个单词
- ‌**Alt + B**‌（或 ‌**Esc + B**‌）：向左跳过一个单词
- ‌**Ctrl + ←/→**‌（部分终端支持）：按单词移动（前/后）‌‌
