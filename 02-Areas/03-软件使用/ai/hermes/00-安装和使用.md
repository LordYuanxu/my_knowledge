## 使用

解决内网问题
```
# 方式 1：命令行直接指定（临时）
HERMES_WEBUI_HOST=0.0.0.0 HERMES_WEBUI_PASSWORD=你的强密码 ./start.sh

# 方式 2：ctl.sh 后台启动时也支持
HERMES_WEBUI_HOST=0.0.0.0 HERMES_WEBUI_PASSWORD=你的强密码 ./ctl.sh start
```

---

## 卸载

```
# 停止 WebUI（如果用了 ctl.sh）
./ctl.sh stop 2>/dev/null

# 停止 Hermes 网关服务（如果安装过）
hermes gateway uninstall 2>/dev/null
```

## 配置kimicode模型

```bash
hermes config set model.provider custom
hermes config set model.base_url https://api.kimicode.com/v1
# 替换为实际地址 hermes config set model.api_key sk-你的Key hermes config set model.default 模型名称
```
