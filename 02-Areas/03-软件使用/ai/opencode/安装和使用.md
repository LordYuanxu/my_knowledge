# 安装和使用

---
## 安装

```bash
# npm 安装
npm install -g opencode-ai
# 安装skills
npm install -g skills
```

---
## 配置

**用户配置目录:** `Windows: %USERPROFILE%\.opencode`
**用户配置文件:** `%USERPROFILE%\.opencode\opencode.json`

---
## opencode-gui

### 需要webview2运行时
### 环境变量

```powershell
# 获取用户级环境变量
[Environment]::GetEnvironmentVariable("WEBVIEW2_BROWSER_EXECUTABLE_FOLDER", "User")

# 删除用户级环境变量
[Environment]::SetEnvironmentVariable("WEBVIEW2_BROWSER_EXECUTABLE_FOLDER", $null, "User")
```

---
## opencode插件

### opencode-swarm

```bash
# 1. 安装
npm install opencode-swarm --prefix "$env:USERPROFILE\.opencode"
```

配置文件`opencode.json`中

```json
{
  "plugin": ["opencode-swarm"]
}
```
