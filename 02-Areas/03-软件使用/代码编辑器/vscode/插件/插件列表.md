
# 插件

---
## 常用插件列表

- Better Comments: **彩色注释高亮**，区分 TODO​、!​ 警告、*​ 重要、?​ 疑问，让注释更清晰。
- Chinese (Simplified) (简体中文) Language Pack for Visual Studio Code
- Indent Rainbow: **缩进彩色标记**，不同层级缩进用不同颜色，快速看清代码嵌套结构。
- Git History
- Peacock: 根据不同的工作区（Workspace）或文件夹，给整个编辑器的标题栏、侧边栏和标签栏统一设置颜色，一眼就能区分不同项目。
- Prettier: **一键格式化**，支持 JS/TS/HTML/CSS/Vue/React 等，保存自动格式化，统一团队风格
- Project Manager: 一键收藏 / 切换多个项目，不用反复打开文件夹、找路径。
- Error Lens: **错误信息直接显示在代码行旁**，不用 hover 即可看完整报错，快速定位问题。

## 美化和体验

### 图标

- Material Icon Theme: **文件图标美化**，不同文件 / 文件夹类型有专属图标，辨识度高。

### 主题 

- One Dark Pro: 经典深色主题，护眼、对比度适中，长期编码舒适。

### 编程语言相关

#### javascript

- ESLint: **JS/TS 语法与质量检查**，实时标红错误、不规范写法，配合 Prettier 自动修复

#### Python

- Python: Python 语法高亮、智能提示、调试、格式化、Linting，Python 开发基础包。
- Pylance: Python 高性能语言服务，提供**更快的智能补全、类型检查、重构**。

#### Go

- Go 语言支持，代码补全、调试、测试、格式化、依赖管理。
