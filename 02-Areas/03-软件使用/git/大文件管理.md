#  Git LFS

---

## 安装

```bash
# macOS
brew install git-lfs
git lfs install

# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y git-lfs
git lfs install
```

---

## 使用

```bash
# 追踪特定类型的大文件
git lfs track "*.png"
git lfs track "*.jpg"
git lfs track "*.pdf"
git lfs track "*.mp4"

# 生成 .gitattributes 文件（记录追踪规则）
git add .gitattributes
git commit -m "Add Git LFS tracking"
```
