# 设计
## mongo新增字段

### sg_split_specimen表

| 字段名                      | 数据类型   | 示范             | 描述               | 来源   | 备注                                                                                                                                                 |
| ------------------------ | ------ | -------------- | ---------------- | ---- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| `data_center`            | String | ningbo/tianjin | 长存地              | 系统插入 | 1.上海的拆分任务:<br>a.如果长存地为宁波,从本地传输到宁波s3<br>b.如果长存地为天津,先从本地传输到宁波cache,再从宁波cache传输到天津s3<br>2.天津的拆分任务: <br>a.如果长存地为宁波,从本地传输到宁波s3<br>b.如果长存地为天津,从本地传输到天津s3 |
| `analysis_type`          | String | pure_sequence  | 分析类型             | 系统插入 | pure_sequence或者为空<br>上海的拆分任务: 如果是纯分析样本, 根据长存地转移到宁波s3或者天津s3<br>天津的拆分任务: 如果是纯分析样本，自动从本地传输到宁波cache                                                    |
| `path.raw_path_cache`    | String | -              | 宁波raw数据cache路径   | 导表更新 | 如果是纯分析样本，会存在这个路径                                                                                                                                   |
| `path.clean_path_cache`  | String | -              | 宁波clean数据cache路径 | 导表更新 | 多样性或者mirna存在这个路径                                                                                                                                   |
| `path.raw75_path_cache`  | String | -              | 宁波raw75数据cache路径 | 导表更新 | mirna存在这个路径                                                                                                                                        |
| `path.cache_transfer_st` | String | no             | 是否转移成功状态         | 导表更新 | no(未开始)/start(开始转移)/end(转移结束)/failed(转移失败)                                                                                                         |
| `is_delete_cache`        | Int    | 0              | 宁波cache路径是否删除    | 导表更新 |                                                                                                                                                    |
