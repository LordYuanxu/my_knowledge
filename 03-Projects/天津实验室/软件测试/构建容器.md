# 构建容器

---
## 基础容器

```sif
Bootstrap: docker
From: m.daocloud.io/docker.io/library/ubuntu:22.04

%files
    # 将本地编译好的bcl-convert复制到容器中
    /home/ubuntu/app/bioinfo/datasplit_v3/bcl-convert/current/usr/bin/bcl-convert /usr/local/bin/bcl-convert
    
%post
    # 更新并安装必要工具
    apt-get update
    # 安装时区数据库
    apt-get install -y tzdata
    
    # 设置为北京时间
    ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
    echo "Asia/Shanghai" > /etc/timezone

%environment
    # 设置环境变量
    export LC_ALL=C.UTF-8
    export LANG=C.UTF-8

%runscript
    # 默认运行脚本
    echo "这是拆分软件集合的singularity容器"
```

---

## 构建

```bash
# 构建容器
sudo /home/ubuntu/app/bioinfo/datasplit_v3/miniconda3/bin/singularity build base.sif base.def
# 测试运行
singularity exec base.sif bcl-convert --version
# 如果报错缺少 .so 库，查看依赖
singularity exec base.sif ldd /usr/local/bin/bcl-convert

```

---

## 进入沙盒模式调整

```bash
# 从已有 SIF 构建沙盒目录
singularity build --sandbox mysandbox/ base.sif
# 进入沙盒
singularity shell --writable mysandbox/
# 将沙盒重新打包为 SIF
sudo singularity build new_base.sif mysandbox/
```
