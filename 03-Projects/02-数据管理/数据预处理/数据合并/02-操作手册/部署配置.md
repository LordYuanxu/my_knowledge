## S3配置

### S3输出路径格式

```
s3nb4://datasplit/{年份}/{模块}/{地点}/{日期}/{批次}/

示例:
s3nb4://datasplit/2026/DataProcessMerge/shanghai/20260326/FX2026032600711/
```

### S3数据源

合并任务需要从S3下载源数据：
- 数据源路径记录在 `merge_samples` 的各个 `*_path` 字段中
- 支持从 LibrarySplit 等模块的输出路径下载

## 常见问题配置

### 磁盘空间不足

**症状**: 任务失败，日志显示磁盘空间不足

**解决**:
```bash
# 检查磁盘空间
df -h /mnt/ilustre

# 清理历史工作目录
find /mnt/ilustre/isanger_workspaceDatasplitV3 -type d -mtime +30 -exec rm -rf {} \;
```

### S3下载失败

**症状**: to_file阶段失败，无法下载S3数据

**检查**:
```bash
# 检查S3连接
aws s3 ls s3nb4://datasplit/

# 检查具体文件是否存在
aws s3 ls s3nb4://datasplit/path/to/file.fastq.gz
```
