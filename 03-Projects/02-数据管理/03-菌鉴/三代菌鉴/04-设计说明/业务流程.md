## 阶段一：数据准备 (to_file)

### 1.1 文件匹配逻辑

外包测序上传的文件中包含 `majorbio_number` 和 `sample_name` 信息（从文件名中提取）。

**文件名解析规则**:
```
示例文件名: S22604013025-RLC1301763-2.consensus.fasta
                    │
                    └── majorbio_number: RLC1301763-2

示例文件名: RLC1301763-2--S7.ab1
                    │      │
                    │      └── sample_name: S7
                    └── majorbio_number: RLC1301763-2
```

### 1.2 构建的文件信息

生成 `third_strain_identification_info.json` 文件：

```json
{
    "RLC1301763-2--S7": {
        "coverage_png": "/mnt/clustre/.../S22604013025-RLC1301763-2.coverage.png",
        "consensus_fasta": "/mnt/clustre/.../S22604013025-RLC1301763-2.consensus.fasta",
        "sample_id": "RLC1301763-2--S7",
        "fastq": "",
        "bam": "/mnt/clustre/.../S22604013025-RLC1301763-2.bam",
        "var_file": "/mnt/clustre/.../S22604013025-RLC1301763-2.var.xls",
        "length_peak_png": "/mnt/clustre/.../S22604013025-RLC1301763-2_read_length_peak.png",
        "ab1_file": "/mnt/clustre/.../S22604013025-RLC1301763-2_1.ab1"
    },
    "RLC1301762-2--S6": {
        "coverage_png": "/mnt/clustre/.../S22604013024-RLC1301762-2.coverage.png",
        "consensus_fasta": "/mnt/clustre/.../S22604013024-RLC1301762-2.consensus.fasta",
        "sample_id": "RLC1301762-2--S6",
        "fastq": "",
        "bam": "/mnt/clustre/.../S22604013024-RLC1301762-2.bam",
        "var_file": "/mnt/clustre/.../S22604013024-RLC1301762-2.var.xls",
        "length_peak_png": "/mnt/clustre/.../S22604013024-RLC1301762-2_read_length_peak.png",
        "ab1_file": "/mnt/clustre/.../S22604013024-RLC1301762-2_1.ab1"
    }
}
```

**文件说明**:
| 文件类型 | 说明 | 用途 |
|---------|------|------|
| `ab1_file` | 原始测序数据(ab1格式) | 结果打包 |
| `consensus_fasta` | 一致序列 | BLAST比对 |
| `coverage_png` | 覆盖度图 | 结果展示 |
| `length_peak_png` | 读长分布图 | 结果展示 |
| `var_file` | 突变位点文件 | 污染判定 |
| `bam` | 比对文件 | 备选：转fastq用于Kraken |
| `fastq` | 原始测序数据(fastq格式) | Kraken分类（优先） |

## 阶段二：分析流程 (Workflow)

### 2.1 流程步骤

```
开始
  │
  ▼
┌────────────────┐
│ 1. 污染判定     │◀── 根据突变位点(var文件)判断样本是否污染
│   (var_judge)  │    - 正常：继续分析
└───────┬────────┘    - 污染：标记quality_rank
        │
        ▼
┌────────────────┐
│ 2. BLAST比对   │◀── 使用consensus_fasta文件
│ (blast_consensus)   与NCBI数据库比对
└───────┬────────┘
        │
        ▼
┌────────────────┐
│ 3. Kraken分类  │◀── 使用fastq文件
│    (kraken)    │    如果fastq不存在但bam存在，将bam转为fastq
└───────┬────────┘
        │
        ▼
┌────────────────┐
│ 4. 结果打包    │◀── 生成result.zip
│  (zip_result)  │
└───────┬────────┘
        │
        ▼
      结束
```

### 2.2 污染判定逻辑

```
var文件分析
    │
    ├───▶ 提取突变位点
    │
    ├───▶ 判断标准：
    │       - 杂合位点比例
    │       - 突变位点分布
    │
    └───▶ 输出 quality_rank:
            - "正常" → 继续后续分析
            - "污染" → 标记污染，可能跳过部分分析
```

### 2.3 BLAST比对逻辑

```
consensus.fasta
      │
      ▼
┌─────────────┐
│  BLASTN    │  ──▶ 比对到NCBI NT数据库
│   比对      │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  结果解析   │  ──▶ 提取topN比对结果
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  写入Mongo  │  ──▶ strain_identification_sample_detail_third表
│  (detail)   │
└─────────────┘
       │
       ▼
┌─────────────┐
│  更新样本表 │  ──▶ strain_identification_sample_third表的taxonomy字段
└─────────────┘
```

### 2.4 Kraken分类逻辑

```
输入文件判断
    │
    ├───▶ fastq文件存在？
    │       ├── 是 → 直接使用fastq
    │       └── 否 → 检查bam文件
    │                   ├── 存在 → bam转fastq
    │                   └── 不存在 → 报错/跳过
    │
    ▼
┌─────────────┐
│  Kraken2   │  ──▶ 物种分类
│  分类分析   │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  结果解析   │  ──▶ 提取物种分类信息
└─────────────┘
```

## 阶段三：结果输出

### 3.1 结果包结构

```
result.zip
├── blast.result.xls              # BLAST比对汇总结果
├── identify.xls                  # 菌鉴报告
├── Data/                         # 数据文件目录
│   ├── RLC1301762-2--S6.ab1
│   ├── RLC1301762-2--S6.consensus.fasta
│   ├── RLC1301762-2--S6.coverage.png
│   ├── RLC1301762-2--S6.read_length_peak.png
│   ├── RLC1301762-2--S6.var.xls
│   ├── RLC1301763-2--S7.ab1
│   ├── RLC1301763-2--S7.consensus.fasta
│   ├── RLC1301763-2--S7.coverage.png
│   └── RLC1301763-2--S7.read_length_peak.png
└── NCBI/                         # BLAST详细结果
    ├── RLC1301762-2--S6.blast.xls
    └── RLC1301763-2--S7.blast.xls
```

### 3.2 输出文件说明

| 文件 | 来源 | 说明 |
|------|------|------|
| `blast.result.xls` | BLAST工具输出整理 | 所有样本的比对汇总 |
| `identify.xls` | 系统生成 | 菌种鉴定最终报告 |
| `Data/*.ab1` | 原始数据 | 原始测序峰图文件 |
| `Data/*.consensus.fasta` | 原始数据 | 一致序列 |
| `Data/*.coverage.png` | 原始数据 | 测序覆盖度图 |
| `Data/*.read_length_peak.png` | 原始数据 | 读长分布图 |
| `Data/*.var.xls` | 原始数据 | 突变位点信息 |
| `NCBI/*.blast.xls` | BLAST工具 | 详细比对结果 |

## 关键决策点

| 决策点 | 判断条件 | 分支处理 |
|-------|---------|---------|
| 污染判定 | var文件分析 | 污染 → 标记quality_rank为"污染" |
| fastq获取 | 检查fastq存在性 | 不存在 → bam转fastq |
| 任务状态 | 分析完成性 | 失败 → 标记status为"failed" |
