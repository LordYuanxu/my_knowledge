## 🏗️ 项目架构概览

```
┌─────────────────────────────────────────────────────────────┐
│                      三代菌鉴流程                            │
├─────────────────────────────────────────────────────────────┤
│  输入层  │  外包测序数据(ab1/fastq/bam) → to_file匹配        │
├─────────────────────────────────────────────────────────────┤
│  分析层  │  污染判定 → BLAST比对 → Kraken分类 → 结果打包     │

│  输出层  │  result.zip (blast结果 + 数据文件 + 鉴定报告)     │
├─────────────────────────────────────────────────────────────┤
│  存储层  │  MongoDB (主表/样本表/详情表) + S3对象存储        │
└─────────────────────────────────────────────────────────────┘
```
