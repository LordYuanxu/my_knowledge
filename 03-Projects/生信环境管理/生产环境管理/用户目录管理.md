# mise管理普通软件

---
## 安装mise

```bash
# 自动安装mise
## 设置MISE_INSTALL_DIR替换为你想要的安装目录;MISE_DATA_DIR指定软件的安装目录
MISE_INSTALL_DIR=~/sg-users/xuyuan/bin/ sh <(curl https://mise.run)
https://mise.run | sh
# 配置写入~/.bashrc
echo 'eval "$(~/.local/bin/mise activate bash)"' >> ~/.bashrc

# 下载 mise 的 musl 预编译版本(如果glibc版本过低)
curl -sSfL https://github.com/jdx/mise/releases/download/v2026.3.9/mise-v2026.3.9-linux-x64-musl.tar.gz | tar xz
```
