# 数据库设计

> **说明**: 三代菌鉴流程 MongoDB 表结构定义  
> **最后更新**: 2026-04-07

---

## 数据库概述

| 项目           | 内容                                    |
| -------------- | --------------------------------------- |
| **数据库类型** | MongoDB                                 |
| **数据库名称** | datasplit                               |
| **表数量**     | 3 张                                    |
| **核心表**     | strain_identification_third（任务主表） |

---

## 表关系图

```
┌─────────────────────────────────────────────────────────────────┐
│           strain_identification_third (任务主表)                │
│  ├─ 1个任务对应 N 个样本                                        │
│  └─ _id (主键)                                                  │
└────────────────────────────┬────────────────────────────────────┘
                             │ 1:N
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│       strain_identification_sample_third (任务样本表)           │
│  ├─ 1个样本对应 1 个文件信息(path对象)                          │
│  └─ main_id → 关联主表                                          │
└────────────────────────────┬────────────────────────────────────┘
                             │ 1:N
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│   strain_identification_sample_detail_third (样本详情表)        │
│  ├─ 存储 BLAST 比对结果                                         │
│  └─ main_id → 关联主表                                          │
└─────────────────────────────────────────────────────────────────┘
```

#### BLAST 字段解释

```
q = Query (查询序列)        s = Subject (目标序列)
│                             │
├─ qseqid: 查询序列ID        ├─ sseqid: 目标序列ID
├─ qstart: 查询起始位置      ├─ sstart: 目标起始位置
└─ qend: 查询结束位置        └─ send: 目标结束位置

bitscore: 比对得分 (越高越好)
evalue: 期望值 (越低越好，接近0表示高度显著)
pident: 相似度百分比 (100%表示完全相同)
length: 比对区域长度
mismatch: 错配碱基数
gapopen: 空位开放数
```
