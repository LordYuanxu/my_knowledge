# 维护
## 查看删除日志 
1. 查看etcd中删除任务的运行状态(可以判断删除任务是否在运行)
    ```bash
    # 查看ftm服务部署的节点
    etcdctl get /wpm2/service/ftm --prefix
    etcdctl get /wpm2/service --prefix | grep ftm
    # 通过key:/%v/s3filedelete/running/%v/%v查看具体人物的情况
    etcdctl get /wpm2/s3filedelete/running --prefix
    ```
2. 查看运行删除任务的节点(对应的key中会包含运行的节点)
3. 跳转到对应的节点查看ftm日志(文件夹/var/log/wpm2下)
   ```bash
   cat /var/log/wpm2/ftm.log | grep "\"UniqueID\":\"12304\""
   zcat /var/log/wpm2/ftm* | grep "\"UniqueID\":\"12304\""
   ```
