# 数据库设计

> **说明**: 数据合并流程 MongoDB 表结构定义  
> **最后更新**: 2026-04-08

---

## 数据库概述

| 项目           | 内容                           |
| -------------- | ------------------------------ |
| **数据库类型** | MongoDB                        |
| **数据库名称** | datasplit                      |
| **表数量**     | 2 张                           |
| **核心表**     | data_process_merge（任务主表） |

---

## 表关系图

```
┌─────────────────────────────────────────────────────────────────┐
│           data_process_merge (任务主表)                         │
│  ├─ 1个释放单对应 N 个样本                                      │
│  └─ _id (主键)                                                  │
└────────────────────────────┬────────────────────────────────────┘
                             │ 1:N
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│       data_process_merge_detail (任务样本表)                    │
│  ├─ 1个样本对应 1 个产品信息(product_info)                      │
│  ├─ 1个样本对应 N 个待合并文件(merge_samples)                   │
│  └─ main_id → 关联主表                                          │ 
└─────────────────────────────────────────────────────────────────┘
```

---

## 表结构详情

#### 状态流转图

```
    no (未启动)
      │
      ▼ (投递任务)
   start (运行中)
      │
      ├──► end (成功完成)
      │
      └──► failed (运行失败)
```

## 数据类型说明

| 数据类型                | 用途                   | 示例值        |
| ----------------------- | ---------------------- | ------------- |
| `rna_raw/rna_clean`     | RNA测序原始/ clean数据 | 标准fastq     |
| `dna_raw/dna_clean`     | DNA测序原始/ clean数据 | 多样性产品    |
| `mirna_raw/mirna_clean` | miRNA测序数据          | 含raw75       |
| `bam`                   | 三代测序BAM格式        | 用pbmerge合并 |
