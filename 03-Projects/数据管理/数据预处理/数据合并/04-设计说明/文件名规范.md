# 文件名规范

> **说明**: 数据合并流程支持的各类产品的文件命名规则  
> **最后更新**: 2026-04-08

---

## 文件名命名规则

根据产品类型不同，输出文件命名格式不同：

### rawdata
- **时空转录组相关(根据三代产品信息判断):** {sample_name}_L01_read_1/2.fq.gz
  - 时空转录组Stereo-Seq
  - 时空转录组Stereo-seqFFPE
  - 时空蛋白转录组Stereo-CITE
- 单细胞: {sample_name}_S1_L001_R1/2/3_001.fastq.gz
- 二代多样性: {mongo_id}--{sample_name}.{primer_name}.R1/2.raw.fastq.gz
- 三代多样性: {mongo_id}--{sample_name}.{primer_name}.raw.fastq.gz
- 二代非多样性: {mongo_id}--{library_name}--{sample_name}.R1/2.raw.fastq.gz
- 三代非多样性(bam格式): {mongo_id}--{sample_name}.bam
- mirna产品: {mongo_id}--{library_name}--{sample_name}.R1/2.raw.fastq.gz
### cleandata
- 二代多样性: {mongo_id}--{sample_name}.{primer_name}.R1/2.clean.fastq.gz
- 三代多样性: {mongo_id}--{sample_name}.{primer_name}.clean.fastq.gz
- mirna产品: {mongo_id}--{library_name}--{sample_name}.clean.fastq.gz
### raw75
- mirna产品: {mongo_id}--{library_name}--{sample_name}.raw75.fastq.gz


### 字段说明

| 字段           | 说明            | 来源                                   |
| -------------- | --------------- | -------------------------------------- |
| `mongo_id`     | MongoDB主键ID位 | data_process_merge._id                 |
| `sample_name`  | 样本名称        | data_process_merge_detail.sample_name  |
| `library_name` | 文库编号        | data_process_merge_detail.library_name |
| `primer_name`  | 引物名称        | data_process_merge_detail.primer_name  |
