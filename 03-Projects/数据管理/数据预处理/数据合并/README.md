## 🎯 支持的产品类型

| 产品类型         | 数据格式  | 合并工具    | 特殊处理     |
| ---------------- | --------- | ----------- | ------------ |
| **普通非多样性** | fastq     | cat         | 标准合并     |
| **多样性**       | fastq     | cat         | 序列重命名   |
| **单细胞**       | fastq     | cat         | 标准合并     |
| **时空转录组**   | fastq     | cat         | 特殊命名格式 |
| **miRNA**        | fastq     | cat         | 支持raw75    |
| **三代测序**     | fastq/bam | cat/pbmerge | bam用pbmerge |
